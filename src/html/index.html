Features

Setting up some often needed features and little tweaks.

data-scrollto Modul

Adding data-scrollto="#id" to any element, animates to the element on click. Also calculates a sticky header, if present.

Blackout Modul

General function for blacking-out the main-content for overlay or sidepanel actions. activateBlackout(param) will create a blackout which param as additional class. That way you can create different blackout click events. e.g. Close sidepanel or close overlay on click.

data-overlay Modul

Adding data-overlay="id" to any element, will set the .overlay#id to active and close all other overlays. Also disables body scroll.

Grouping overlays with data-group will append arrows to go to the next/prev overlay.

Looping in overlay groups is possible with data-loop="true".

Adding data-close="true" will add an close icon.

Open Overlay Open Overlay without loop

data-toggle ModulWIP

Adding data-toggle="#id" to any element, will set the #id to active.

Accordion

Grouping toggles with data-group will create an accordion style function, that closes all other group members, when another one opens up.

Back to top Modul

Create a back to top arrow, when scrolled down a little. Pretty basic.

Lazy Loading Modul

Add the class lazyload to an img and set the real source within data-src or data-src-retina. If its not an img - the background image will be set instead.

Sidepanel Modul

Sidepanels on the right is implemented. It can be easily adapted to work on the other side as well.

They can be opened by link with data-open-panel="right"

Helper

These helper classes are for backend developers or content editors, who dont want to mess around with SASS. Only use the following helper classes if you are one of those and stay away from them (and use BEM module classes) if you are the frontend developer.

Text

Change the alignment and some general styles of the text.

--text-left --text-center --text-right --text-upper --text-lower --text-bold --text-italic

Float

Please dont.

--float-left --float-right

Hidden

Simple hide something based on the breakpoints.

--hidden-{bp}-up --hidden-{bp}-down

Aspect-ratio

--aspect-ratio --aspect-ratio-16-9
