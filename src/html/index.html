Javascript

Setting some useful default functions and a expandable, modulare js-boilerplate

data-scrollto Modul

Adding data-scrollto="#id" to any element, animates to the element on click. Also calculates a sticky header, if present.

data-overlay ModulWIP

Adding data-overlay="id" to any element, will set the .overlay#id to active and close all other overlays. Also disables body scroll. Grouping overlays with data-group will append arrows to go to the next/prev overlay.
Open Overlay

data-toggle ModulWIP

Adding data-toggle="#id" to any element, will set the #id to active.
